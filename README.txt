To compile the DNS client program, run the following command in cmd "python dnsClient.py <hostname>" without quotation mark, where <hostname> is replace by 5 given host names.

1) Use the following to run the DNS client with hostname

python dnsClient.py www.cnn.com

python dnsClient.py www.gmu.edu

python dnsClient.py www.vt.edu

python dnsClient.py www.youtube.com

python dnsClient.py www.example.com


2) Use the following command to run DNS using domain name

python dnsClient.py cnn.com

python dnsClient.py gmu.edu

python dnsClient.py vt.edu

python dnsClient.py youtube.com

python dnsClient.py example.com
